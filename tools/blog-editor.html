Markdown / 博客可视化编辑器

支持拖图和粘贴截图，自动插入图片语法；可导出文章并一键更新 posts.json。

图片插入

未选目录：仅插入 Markdown，不写入图片文件。

建议图片前缀统一为 `./assets/Blog/文章名`，可先点“创建并切换”自动生成子目录。

按文章文件名创建并切换到图片子目录。

Front Matter

留空表示非加密文章。格式会自动写成 `MI: 提问|编号`。

发布导出

请选择本地 content/blog 目录后再导出（建议私密仓库目录）。

“导出 + 更新索引”会生成与 `update-posts-index.bat` 相同结果的 `posts.json`。

把图片拖到这里，或在编辑框中直接粘贴截图（Ctrl+V）。
