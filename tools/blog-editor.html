Markdown

  1. 先选图片主目录（`assets/Blog`），填文章文件名（如 `P1`）。
  2. 点“创建并切换”，自动创建 `assets/Blog/P1` 并切换。
  3. 拖图到下方编辑区，自动复制并插入 Markdown 图片地址。
  4. 在头文字区填写 `title / MI / hidden` 等信息后导出 MD。
  5. 私密密码在 `ninocat_private/tools/secrets` 编辑；再运行 `update-posts-index.bat` 与 `add-post.bat`。

常用目录与图片

按文章文件名创建并切换到图片子目录。
未选目录：仅插入 Markdown，不写入图片文件。

建议顺序：填文章名 -> 创建并切换 -> 拖图。

头文字

|
摘要会用于博客卡片描述。
可留空，拖入首图后自动填写。

`MI` 留空表示非加密文章；会写成 `MI: 提问|编号`。

导出

选择私密仓库目录目录后再导出。

密码维护：`ninocat_private/tools/secrets`。导出后运行：`update-posts-index.bat` + `add-post.bat`。

把图片拖到这里，或在编辑框中直接粘贴截图（Ctrl+V）。
